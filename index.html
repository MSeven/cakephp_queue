CakePHP Queue Plugin by MSeven

A minimalistic JobQueue Plugin for CakePHP

Dependencies

none

Install

* Copy the files in this directory into /plugins/queue. * Run the following command in the cake console to create the tables: cake schema run create -path plugins\queue\config\sql -name queue
